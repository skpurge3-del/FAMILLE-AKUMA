## 📛 Présentation

**FAMILLE AKUMA** est une page d'avertissement de **ban permanent**.  
Ce projet affiche une bannière stylisée avec un design sombre fusionnant le **noir** et le **jaune**, accompagnée d'un bouton de redirection vers un groupe WhatsApp.

---

## 🎨 Aperçu

- 🔥 Fond animé noir et jaune
- 🖼️ Image `akuma.jpg` en overlay fusionné
- ✨ Particules jaunes flottantes
- 💬 Bouton WhatsApp avec effet glow
- 📱 Responsive (mobile friendly)

---
