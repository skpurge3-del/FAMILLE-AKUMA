## 📜 Description

**FAMILLE AKUMA** est un outil web interactif permettant de générer un email de signalement pré-rempli à destination du support WhatsApp (`support@whatsapp.com`).  
Idéal pour signaler rapidement des numéros impliqués dans :

- 🛑 Spam / Harcèlement
- 🛑 Usurpation d'identité
- 🛑 Comptes frauduleux
- 🛑 Violation des conditions d'utilisation

## ⚙️ Utilisation

1. **Placer l'image `akuma.jpg`** dans le même répertoire que le fichier HTML.
2. Ouvrir `index.html` dans un navigateur (Chrome, Firefox, Edge, etc.).
3. Remplir le numéro WhatsApp et le motif.
4. Cliquer sur **ENVOYER BAN REPORT**.
5. Le client mail s'ouvre automatiquement. Envoyer le message.
